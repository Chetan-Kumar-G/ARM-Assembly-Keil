# 💻 Requirements

- Keil µVision 5
- ARM7 / LPC2148 (or compatible ARM processor)
- Basic knowledge of ARM Assembly Language

---

# ▶️ How to Run

1. Open **Keil µVision**
2. Create a new ARM project.
3. Add one of the `.s` assembly files.
4. Build the project.
5. Run using the simulator or supported hardware.
6. Observe register values in the Registers window.
